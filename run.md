# IssueFlow Application

## Getting Started

1. **Database**: Make sure Docker is running and execute:
   `docker-compose up -d`
2. **Install Dependencies**:
   `npm install`
3. **Environment**: Create a `.env` file based on `.env.example`.
4. **Run Development**:
   `npm run start:dev`

## Running Tests

- **Unit Tests**: `npm run test`
- **E2E Tests**: `npm run test:e2e`
